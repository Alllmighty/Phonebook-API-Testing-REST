# 🌐 Phonebook API Automation (REST Assured)

A robust API testing framework built with **Java** and **Rest Assured**, focused on validating RESTful services for user management.

## 🚀 Key Features
- **Advanced Specifications**: Centralized `RequestSpecification` and `ResponseSpecification` for reusable test configurations.
- **Dynamic Data Extraction**: Implemented logic to fetch and validate data across multiple API pages (Pagination support).
- **Serialization (DTO)**: Used **Lombok POJOs** for automated JSON-to-Object mapping (Serialization/Deserialization).
- **Comprehensive Validation**: Includes Regex-based email pattern matching and precise timestamp verification using `OffsetDateTime`.
- **Reporting**: Full integration with **Allure Reporting** using Epics, Features, and detailed Steps.

## 🛠️ Tech Stack
- **Java 17+**
- **Rest Assured** (Core framework)
- **TestNG** (Test runner)
- **Lombok** (Data management)
- **Allure** (Professional reporting)
